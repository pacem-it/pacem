# `<pacem-binder>`

`PacemBinderElement` visually connects two items with a beziér curve.  
The connected items can be both **Elements** and **Points** (`Pacem.Point` has type `{x: number, y: number}`).

## Customizations

Properties available allow to customize the aspect and the behavior of a binder:

- `fromAnchor`: where to force the anchor on the `from` Element (ignored for points). Can be `top`, `right`, `bottom`, `left`, `auto` (default).
- `toAnchor`: same as _fromAnchor_, but for the `to` element.

**Custom styling** can be obtained both via `cssClass` and `color` property.

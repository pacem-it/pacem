## Dynamic Icons

`PacemIconElement` allows to dynamically render icons onto the screen.  
The default iconset in Pacem JS is the [Material IconSet](https://material.io/resources/icons), which you can plainly use exploiting the default
embedded themes:

```html
<!-- exploits ligatures -->
<i class="pacem-icon">sentiment_very_satisfied</i>
```

## Material, FontAwesome, URLs, ...
The same above result can be obtained using the `<pacem-icon>` element,
which ultimately defaults to the material icon set:

```html
<pacem-icon icon="sentiment_very_satisfied"></pacem-icon>
```

Nevertheless you can include [FontAwesome library](https://fontawesome.com/) and, similarly:

```html
<pacem-icon icon="far fa-laugh"></pacem-icon>
```

URLs are also welcome.

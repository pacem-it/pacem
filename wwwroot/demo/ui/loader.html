## Spinners

`PacemLoaderElement` is a spinner UI webcomponent that hints an invisible task
running in background.

It gets activated by toggling its watchable `active` property/attribute.

## Overlap

Le loader can be overlapped to any element by simply using css classes:

- Use **pacem-relative** for the containing element.
- Include **pacem-size** (plus **size-lg**, ...) in the containing element to ensure a minimum height.
- Apply **pacem-hover** to the loader. 

Please lurk into the code-snippet above to see it in action.

> This is not restricted to loaders.
